Web Design

Introducing "UofT Timetable Builder," a cutting-edge web application created to streamline the process of course selection and schedule organization for students at the University of Toronto. This powerful tool is designed to help students efficiently arrange their course schedules, optimizing their time management and ensuring their academic and personal commitments are perfectly balanced.

The UofT Timetable Builder features an intuitive interface that allows students to easily search for courses, view class timings, and avoid scheduling conflicts. With a few clicks, students can generate a personalized timetable that fits their preferences, such as early or late start times, short or long intervals between classes, and reasonable walking time between different campus locations.

One of the standout features of the UofT Timetable Builder is its ability to provide students with multiple schedule options. This flexibility enables students to compare different timetables and select the one that best suits their needs, enhancing their ability to manage their academic workload alongside other responsibilities.

Moreover, the application includes advanced filters and preferences, allowing students to customize their schedules based on various criteria. Whether it's minimizing the number of days spent on campus or ensuring that all classes are grouped together to reduce downtime, the UofT Timetable Builder caters to the diverse needs of the student body.

Data Analysis

The project "Identifying Opportunities: Accelerating SDG Progress" is a comprehensive data analysis initiative aimed at advancing the Sustainable Development Goals (SDGs). This project leverages advanced data analytics to identify key opportunities and barriers in achieving SDG targets, thereby accelerating progress toward a sustainable future.

Utilizing a robust dataset, the analysis involves multiple statistical methods and visualization techniques to uncover insights about the relationship between various socio-economic factors and SDG outcomes. The project employs tools like histograms and scatter plots to illustrate the distribution and correlation of data points, making it easier to understand complex relationships.

One of the primary focuses of this analysis is the comparison of gender equality metrics across different regions. Histograms are used to display the distribution of gender-related industry differences across continents such as Europe, Asia, and Africa. These visualizations help in understanding the disparities and progress in gender equality in different parts of the world.

The project is meticulously planned and executed, starting from data collection and cleaning to detailed statistical analysis and interpretation. The use of modern data visualization tools ensures that the findings are not only accurate but also accessible to a broad audience, including policymakers, researchers, and the general public.

Cloud Computing

The project "CloudHub" is a cloud storage feature designed for use within the Academia Imperial project. It provides users with the capability to store files securely in the cloud. Built on Amazon Web Services (AWS), CloudHub offers a reliable and secure environment for uploading, downloading, and managing files, making it an ideal solution for academic research and project collaboration.

CloudHub leverages AWS S3 to store files securely and durably. Users can easily upload and download files of various types and sizes, ensuring that all data is safely stored and accessible from anywhere. AWS S3's high availability and scalability ensure that the platform can handle increasing amounts of data while maintaining excellent performance.

The platform's user interface is built with AWS Amplify, providing a smooth and responsive user experience. Amplify simplifies the integration of various AWS services and enables quick deployment of the application, ensuring that CloudHub remains flexible and adaptable to the needs of the Academia Imperial project.
